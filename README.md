# 발란 프론트엔드 과제

발란 프론트엔드 사전과제 프로젝트 입니다.

[과제 배포 링크](https://seungwoo321.github.io/balaan/)

## 개발환경

- node.js v16.15.0
- Vue 2.7 + vite
- EsList: Standard Rule +@

## 설치 및 실행

### 의존성 패키지 실행

```bash
npm install
```

### 개발 환경 실행

```bash
npm run dev
```

기본 <http://localhost:5173/balaan> 로 접속

### 프로덕션 환경 실행

```bash
### 빌드
npm run build 

## 실행
npm run preview
```

기본 <http://localhost:4173/balaan> 로 접속
